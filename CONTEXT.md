# みんなの集金

クローズドコミュニティ向けのイベント出欠管理・集金ツールのドメイン用語集。

## Language

**コミュニティ**:
イベントを束ねる運営単位。1人のオーナーが0件以上のコミュニティを所有する。
_Avoid_: グループ, サークル, 団体

**主催者**:
コミュニティを所有し、イベント作成・参加者管理・集金状況管理を行う認証済みユーザー。
_Avoid_: 管理者, 運営者

**所有コミュニティ**:
ユーザーがオーナーとして現在所有しているコミュニティ。削除済みのコミュニティは所有コミュニティ数に含めない。
_Avoid_: 作成済みコミュニティ, 累計コミュニティ

**コミュニティ作成上限**:
1人のユーザーが同時に所有できるコミュニティ数の上限。現在の上限は5件。
_Avoid_: 累計作成上限

## Example Dialogue

開発者: このユーザーは過去に6件のコミュニティを作っていますが、2件は削除済みです。新しく作成できますか？

ドメイン担当: はい。所有コミュニティは4件なので、コミュニティ作成上限の5件にはまだ達していません。
